# sloppy

## できること

台湾版のガンダムオンラインのログを、即時あやしい日本語に翻訳出来ます。

最前面表示でゲーム画面の上に重ねられます。

独自の変換フィルターを作成できます。

自軍の指揮系コマンドのクールタイムを計算できます。
